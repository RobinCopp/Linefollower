## specifications

microcontroller: Arduino Nano imitatie (ATmega 328P)

motors: Micro Metal Gearmotors (6V 50:1)

h-bridge: DRV8833

sensors: QTR-8A

batteries: 18650 li-ion (2x)

wireless communication: HC-05

distance sensor - motors: 85 mm

weight: 231 g

speed: 0,7 m/s
